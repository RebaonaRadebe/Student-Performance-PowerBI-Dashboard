# 📊 Student Performance Dashboard

## Overview

This project is an interactive Power BI dashboard that analyzes student performance across different modules. It provides key insights into student results using KPI cards, charts, and filters.

## Dashboard Features

* Overall Average Mark KPI
* Total Students KPI
* Failed Students KPI
* Average Student Performance by Module (Bar Chart)
* Pass vs Fail Distribution (Pie Chart)
* Module Filter (Slicer)

## Tools Used

* Microsoft Power BI
* Microsoft Excel

## Skills Demonstrated

* Data visualization
* Dashboard design
* KPI creation
* Data filtering with slicers
* Creating and formatting charts
* Basic DAX measures
* Report formatting and customization
